# Orbital-Sim
This is a set of private python simulation code for calculating orbital periods for satellites based on altitude. It is meant for personal learning and experimentation in aerospace physics.
# What it Does
- Calculates orbital periods for CIRCULAR satellite orbits
- Implements Newton's Laws of Gravitation
- Models the orbits
# Key Concepts
- Orbital Mechanics
- Circular Velocity and Period
- Python programming for computations
- Aerospace Fundamentals
